# Support

## Getting help

- Check [README.md](README.md) for usage, examples, and troubleshooting.
- Check [CONTRIBUTING.md](CONTRIBUTING.md) for local development and quality checks.

## Before opening an issue

Please include:

- Python version
- OS
- exact command or API snippet
- expected output
- actual output
- minimal reproducible markdown sample

## Issue categories

- bug report (incorrect or unexpected redaction)
- rule enhancement request
- performance optimization request
- documentation improvement
